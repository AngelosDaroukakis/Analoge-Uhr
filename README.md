# Analoguhr

*Eine interaktive analoge Uhr, die Stunden-, Minuten- und Sekundenzeiger mithilfe von JavaScript in Echtzeit bewegt.*

## Überblick
Dieses Projekt stellt eine funktionale analoge Uhr dar, die mit **HTML**, **CSS** und **JavaScript** umgesetzt wurde. Die Zeiger basieren auf der aktuellen Systemzeit und werden über CSS-Transformationen animiert. Ziel ist es, das Zusammenspiel von DOM-Manipulation, Zeitlogik und Animation verständlich darzustellen.  
Aktueller Status: **fertig**.

## Installation
1. Lade das Projekt herunter oder klone das Repository.  
2. Lege die drei Dateien **index.html**, **style.css** und **analogClock.js** in dasselbe Verzeichnis.  
3. Öffne die Datei `index.html` im Browser oder starte einen Live-Server.
